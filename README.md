## 環境構築

### 0. 前提条件

- Python>=3.11
- uv

### 1. リポジトリのクローン

```bash
git clone https://github.com/karakuri-ai/wandb-fully-connected-2025-karakuri-workshop.git
cd wandb-fully-connected-2025-karakuri-workshop
```

### 2. 依存関係のインストール

```bash
uv sync --all-groups
```

### 3. 環境変数の設定

```bash
cp env.example .env
```

以下の値を設定してください。

- OPENAI_API_KEY
- WANDB_API_KEY
- WEAVE_PROJECT_NAME
